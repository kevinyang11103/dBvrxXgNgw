# 社团管理系统 java1067

社团管理系统是一款基于前后端分离架构的软件，旨在为社团的日常管理提供便捷服务。以下是该系统的技术栈和功能模块介绍。

## 技术栈

- **前端技术**：Vue、Element-UI
- **后端技术**：Spring Boot、MyBatis Plus
- **数据库**：MySQL
- **开发语言**：Java

## 功能模块

系统分为两个角色：管理员后台和社团成员后台，各自提供不同的功能模块。

### 管理员后台

- **登录模块**
- **用户管理**：管理所有用户的账户信息。
- **社团管理**：对社团进行增删改查等操作。
- **社团类型管理**：分类管理社团类型。
- **社团成员管理**：管理社团成员信息。
- **通知信息管理**：发布和管理通知信息。
- **入团申请管理**：处理成员的入团申请。
- **费用管理**：管理社团相关费用。

### 社团成员后台

- **社团信息管理**：查看和管理社团的基本信息。
- **社团成员管理**：社团内部成员管理。
- **入团申请**：提交和查询入团申请状态。
- **社团活动管理**：发布和管理社团活动。
- **通知信息管理**：接收和查看社团通知。

## 系统角色

- **管理员**：负责整个系统的管理和维护。
- **社团成员**：参与社团活动，管理自己的社团事务。

## 运行环境

- **服务器**：推荐使用支持Java的应用服务器，如Tomcat。
- **数据库**：MySQL 数据库服务器。
- **客户端**：现代浏览器。
